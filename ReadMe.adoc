= Project: Smart Home Intrusion Detection System

An IoT-based security alerting system that detects motion via PIR sensor and delivers real-time notifications to a Telegram bot over a secure MQTT pipeline.

== Step 1: Installation

=== Prerequisites

Install the following Arduino libraries via the Library Manager:

1. `PubSubClient` — MQTT messaging
2. `Grove RGB LCD` — LCD display
3. `WiFiS3` — WiFi connectivity for Arduino UNO R4

On the Raspberry Pi, install Python dependencies:

 pip install paho-mqtt requests

=== Credentials Setup

1. Copy `arduino_secrets.h.example` to `arduino_secrets.h`
2. Fill in your WiFi network name and password
3. On the Raspberry Pi, set your Telegram credentials as environment variables:

 export BOT_TOKEN="your_telegram_bot_token"
 export CHAT_ID="your_chat_id"

NOTE: Never commit `arduino_secrets.h` to version control. It is listed in `.gitignore`.

== Step 2: Assemble the Circuit

Connect components to the Arduino UNO R4 WiFi as follows:

|===
| Component | Pin

| PIR Motion Sensor | D2
| Buzzer | D3
| LED Button | D5
| Grove RGB LCD | I2C (SDA/SCL)
|===

Refer to the circuit diagram in the project documentation for full wiring layout.

== Step 3: Load the Code

1. Open `Intrusion_Detection__jan23a.ino` in Arduino IDE
2. Ensure `arduino_secrets.h` is in the same folder with your credentials filled in
3. Select board: *Arduino UNO R4 WiFi*
4. Upload to the board

Run the Raspberry Pi subscriber script:

 python3 mqtt_subscriber.py

=== How It Works

* PIR sensor detects motion and sends signal to Arduino
* Arduino triggers buzzer, LED, and LCD alert, then publishes MQTT message
* Raspberry Pi receives MQTT message and forwards it to Telegram bot
* Alert delivered to phone in real-time
* System auto-resets to secure state after 10 seconds of no motion
